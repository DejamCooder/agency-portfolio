# 🚀 Dejam Marketing: Website Development

A modern, responsive restaurant management dashboard and menu system. 
Built to help business owners manage reservations and digital menus with ease.

## ✨ Key Features
* **Professional Slider, Dynamic Hero Carousel**: An interactive, touch-friendly slider designed to showcase top-tier branding and high-conversion imagery.
* **Responsive Dashboard**: A sidebar with navigation and analytics tracking.
* **Automated Menu Grid**: A CSS Flexbox-powered layout that keeps food items perfectly aligned across mobile and desktop screens.

## 🛠️ Tech Stack
* **HTML5**: For semantic structure.
* **CSS3**: Using Flexbox for advanced layouts and responsiveness.
* **JavaScript**: For interactive table generation and cart logic.
